---
title: useSize
description: Track the size of an element.
---

## Usage

```tsx twoslash include imports
import { useSize } from '@one-stack/use-size'
```

Observe an element:

```tsx twoslash
// @include: imports
// ---cut---
function MyComponent() {
  const { ref, width, height } = useSize()

  return <div ref={ref}>...</div>
}
```

Provide default values:

```tsx twoslash
// @include: imports
// ---cut---
const { ref, width = 0, height = 0 } = useSize()
```
