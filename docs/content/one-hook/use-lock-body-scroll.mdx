---
title: useLockBodyScroll
description: Lock and unlock the body scroll.
---

## Usage

```tsx twoslash include imports
import { useLockBodyScroll } from '@one-stack/use-lock-body-scroll'
```

The body scroll is automatically unlocked when the hook unmounts.

```tsx twoslash
// @include: imports
// ---cut---
const { lock, unlock, locked } = useLockBodyScroll()
```

Lock on mount:

```tsx twoslash
import { useEffect } from 'react'
// @include: imports
// ---cut---
const { lock } = useLockBodyScroll()

useEffect(lock, [lock])
```
