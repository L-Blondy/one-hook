## Installation

```package-install
npm install @1hook/use-document-has-focus
```

## Usage

```tsx twoslash include imports
import { useDocumentHasFocus } from '@1hook/use-document-has-focus'
```

Track the document visibility:

```tsx twoslash
// @include: imports
// ---cut---
const hasFocus = useDocumentHasFocus()
```
